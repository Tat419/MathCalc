# 📐 Математический калькулятор

Простой и удобный консольный калькулятор на Python для решения школьных и базовых инженерных математических задач. Поддерживает работу с уравнениями, прогрессиями, геометрией и тригонометрией.

---

## ✨ Возможности

### 1. Квадратные уравнения
- Вычисление дискриминанта
- Поиск корней (два, один или отсутствие действительных решений)

### 2. Прогрессии
- **Арифметическая прогрессия**: нахождение n-го члена и суммы первых n членов
- **Геометрическая прогрессия**: нахождение n-го члена и суммы первых n членов (с учётом случая `q = 1`)

### 3. Площади двумерных фигур (2D)
| Фигура | Доступные методы расчёта |
|--------|--------------------------|
| Треугольник | По основанию и высоте, по 3 сторонам (Герон), по двум катетам, по двум сторонам и углу |
| Прямоугольник | По длине и ширине |
| Квадрат | По стороне |
| Ромб | По диагоналям / по стороне и высоте |
| Параллелограмм | По основанию и высоте |
| Трапеция | По двум основаниям и высоте |
| Круг | По радиусу |

### 4. Объёмы и площади трёхмерных тел (3D)
| Тело | Полная поверхность | Боковая поверхность | Объём |
|------|:------------------:|:-------------------:|:-----:|
| Прямоугольный параллелепипед | ✅ | ✅ | ✅ |
| Куб | ✅ | ✅ | ✅ |
| Цилиндр | ✅ | ✅ | ✅ |
| Конус | ✅ | ✅ | ✅ |
| Шар | ✅ | — | ✅ |
| Прямая призма | ✅ | ✅ | ✅ |
| Правильная пирамида | ✅ | ✅ | ✅ |

### 5. Тригонометрия
- Вычисление `sin`, `cos`, `tg`, `ctg` по углу в градусах
- Автоматическая проверка на неопределённость (деление на ноль при кратных 90°)
